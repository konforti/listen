listen
======
Inline Audio Player

## Demo
http://www.konforti.net/listen/examples/

## Usage
```html
<!DOCTYPE html>
<html>
  <head>
    <script src="listen.js"></script>
  </head>
  <body>
    <span class="listen-node" data-src="audio.mp3" data-start="4" data-end="13">That's one small step for [a] man, one giant leap for mankind.</span>
  </body>
</html>
```
